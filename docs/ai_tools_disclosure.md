# AI Tools Disclosure

Some code and documentation for this project were drafted or revised with AI assistance.

AI tools were used to help:

- organize the public GitHub repository structure
- draft and debug parts of the Streamlit interface
- format the app-ready prediction CSV
- prepare documentation files
- clarify the relationship between the backend model pipeline and the Streamlit app output

All AI-assisted code and documentation should be reviewed, tested, and edited by the student before public sharing, presentation, or submission.

The student is responsible for understanding:

1. how the model pipeline notebook creates prediction outputs
2. how `data/app_predictions.csv` is structured
3. how `app.py` reads and displays the prediction output
4. why the repository includes a public-health disclaimer
5. why private credentials and secrets should not be uploaded to a public repository

This project is an experimental student research prototype and is not an official public-health advisory system.
